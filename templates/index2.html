This is what Lottie Images are

Lottie is a lightweight, open-source animation format developed by Airbnb that allows developers to render high-quality animations in real time on web and mobile applications. These animations are exported as JSON files from tools like Adobe After Effects using the Bodymovin plugin, making them resolution-independent, scalable, and small in size. Lottie animations can be easily embedded into websites or apps and controlled with JavaScript, allowing for interactivity like play on click, loop, or animation syncing with scroll. This makes Lottie an ideal choice for modern UI/UX design, enhancing user engagement without the performance cost of traditional GIFs or videos.
